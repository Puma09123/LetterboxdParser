# Letterboxd Parser

## Постановка задачи

Цель проекта — реализовать парсер сервиса Letterboxd, который по уникальному
идентификатору пользователя собирает информацию о фильмах и оценках,
выставленных этим пользователем.

Полученные данные могут быть использованы в дальнейшем для анализа
предпочтений пользователя или интеграции в рекомендательную систему фильмов,
основанную на данных из нескольких источников.

## Описание

Парсер получает имя пользователя Letterboxd, загружает страницу с его
оценками и извлекает информацию о фильмах и пользовательских рейтингах.
