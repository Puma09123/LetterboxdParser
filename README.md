# Letterboxd Parser

## Постановка задачи

Цель проекта — реализовать парсер сервиса Letterboxd, который по уникальному идентификатору пользователя собирает информацию о фильмах и оценках, выставленных этим пользователем.

Полученные данные могут быть использованы для анализа пользовательских предпочтений, а также в дальнейшем могут служить основой для интеграции в рекомендательную систему фильмов, основанную на данных из нескольких источников.

## Описание

Парсер принимает имя пользователя Letterboxd, загружает страницу с его фильмами и оценками и извлекает информацию о названиях фильмов и пользовательских рейтингах.
