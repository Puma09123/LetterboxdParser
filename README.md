# Letterboxd Parser

## Постановка задачи
Цель проекта — реализовать парсер сервиса Letterboxd, который по уникальному идентификатору пользователя собирает информацию о фильмах и оценках, выставленных этим пользователем. Полученные данные могут быть использованы для анализа предпочтений пользователя или интеграции в рекомендательную систему фильмов, основанную на данных из нескольких источников.

## Описание решения
Парсер получает имя пользователя Letterboxd, загружает страницу с его фильмами и оценками, после чего извлекает информацию о названиях фильмов и пользовательских рейтингах. Проект реализован с помощью функций `get_user_ratings(username)` — собирает список фильмов и оценок пользователя, и `save_to_json(data)` — сохраняет результаты в JSON-файл. Функции реализованы как отдельные, модульные блоки, что позволяет использовать их в расширенных системах анализа и рекомендации фильмов, легко добавлять новые источники данных или расширять функционал, а также передавать результаты в аналитические алгоритмы, базы данных или другие программы.

## Пример использования, установка зависимостей и выходные данные

```python
# Установка зависимостей
# pip install requests beautifulsoup4

from parser import get_user_ratings, save_to_json

# Пример запуска парсера
username = "letterboxd_username"  # замените на актуальный ID пользователя
films = get_user_ratings(username)

# Сохраняем результаты в JSON
save_to_json(films)

print("Данные сохранены в файл results.json")

# Пример структуры выходных данных (results.json):
# [
#   {"title": "Fight Club", "rating": "4.5"},
#   {"title": "The Matrix", "rating": "5"},
#   {"title": "Inception", "rating": "4"}
# ]
```
